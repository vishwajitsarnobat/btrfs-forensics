1. Running at once:
uv run --python 3.14 python main.py sandbox.img -o recovery_output
uv run --python 3.14 python -m unittest discover -s tests -v

2. Individual tests:
uv run --python 3.14 python -m unittest tests.test_crc32c -v
uv run --python 3.14 python -m unittest tests.test_inode_parser -v
uv run --python 3.14 python -m unittest tests.test_integration -v

3. Mount and unmount:
sudo mount -o loop,ro -t btrfs sandbox.img mnt_sandbox
sudo umount mnt_sandbox
